# 幸运花圃

一个小型回合制种花积分游戏。

## 运行方式

用浏览器直接打开 `index.html` 即可开始游戏（无需服务器）。

## 玩法简述

- **棋盘**：5×5，正中央为雕像，其余 24 格为空位。
- **仓库**：初始拥有红玫瑰、白百何、蹦蹦灵；点击可查看名称、分类与效果。
- **种花**：每回合从仓库随机抽取若干种植物（最多 24 个），随机放入空位，按效果结算积分；结算后从三个随机种植物中选一个加入仓库。
- **兑换币**：每完成 5 回合获得 1 枚，可在「格子商店」购买格子效果，购买后从下一回合开始生效。

祝你在花圃里玩得开心。
